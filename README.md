# Stat Priority Note

[![CurseForge](https://img.shields.io/badge/CurseForge-Download-orange)](https://legacy.curseforge.com/wow/addons/stat-priority-note) [![CurseForge Downloads](https://img.shields.io/curseforge/dt/1477150)](https://legacy.curseforge.com/wow/addons/stat-priority-note) [![MIT License](https://img.shields.io/badge/license-MIT-blue.svg)](LICENSE)

A lightweight World of Warcraft addon that displays stat priorities for your current class and specialization directly above the Character window.

The goal of this addon is to provide a quick reminder of stat priorities while inspecting gear, without needing to open guides or websites.

Stat priority data is based on the current **Wowhead class guides**.  
Last data update: **2026-03-04**

***

## Features

*   Displays stat priority based on your **current specialization**
*   **ElvUI styling** when ElvUI is installed
*   Lightweight and performance-friendly
*   Automatically updates when changing specialization

***

## Installation

### CurseForge

Install directly from CurseForge:

[https://legacy.curseforge.com/wow/addons/stat-priority-note](https://legacy.curseforge.com/wow/addons/stat-priority-note)

### Manual Installation

1.  Download the latest release
2.  Extract the folder
3.  Move it to: World of Warcraft/retail/Interface/AddOns/

***

## Compatibility

*   **Retail / Midnight (12.0+)**
*   **ElvUI supported** (automatic skin)

***

## Data Source

Stat priorities are derived from **Wowhead class guides** and may change as the meta evolves.
